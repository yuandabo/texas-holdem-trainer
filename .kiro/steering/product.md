# Product Overview

Texas Hold'em Trainer (德州扑克训练器) — a poker training app that teaches Texas Hold'em rules and strategy through interactive gameplay.

## Modes
- **AI Training Mode**: Single-player vs AI opponent ("Luna") with optional hand-rank and win-rate hints to help players learn
- **PVP Mode**: Real-time 1v1 multiplayer via WebSocket rooms with room codes, reconnection support, and action timeouts

## Core Gameplay
- Standard Texas Hold'em: pre-flop → flop → turn → river → showdown
- Betting actions: check, call, raise, fold, all-in
- Chip-based economy with blinds (SB 10 / BB 20), starting stack 2000
- Game ends when one player runs out of chips

## Target Platforms
- WeChat Mini Program (weapp)
- H5 (mobile web)
- Android (via Capacitor)

## Language
- UI text is in Chinese (Simplified)
- Code comments are a mix of Chinese and English
- Variable names, types, and API messages are in English
